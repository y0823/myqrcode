二维码工作室 - 使用指南

欢迎使用二维码工作室，本工具支持高自由度定制的现代化二维码生成以及强大的 REST API 接口。

1. 基本操作

  • 单张生成：在文本框内粘贴 URL、文本，点击生成后即可在右侧预览。点击“下载”或“复制图片”即可提取。
  • 批量生成：在批量输入框内输入多行内容（每行一条），系统会生成对应的多张二维码并打包为 .zip 文件供一键下载。
  • 自定义样式：您可以调整像素大小，使用色彩轮盘修改二维码颜色与背景色，或者点击“选择文件”嵌入自定义中心 Logo。

2. 实用技巧：生成自动连接的 WiFi 二维码

您可以在内容框输入以下格式文本，生成的二维码使用手机原生相机扫描后即可自动连接 WiFi：

WIFI:S:您的WiFi名称;T:WPA;P:您的密码;;

注意：结尾必须带上 ;; 两个分号。

3. 在 Excel 中使用 IMAGE 公式

在较新版本的 Excel (Office 365 / 2024) 中，您可以直接读取 A1 单元格内容并渲染二维码：

=IMAGE("http://localhost:3000/api/generate?text=" & ENCODEURL(A1))

若要自定义颜色（例如蓝色），记得将 # 转义为 %23：

=IMAGE("http://localhost:3000/api/generate?size=200&colorDark=%230000FF&text=" & ENCODEURL(A1))

4. API 接口文档 (面向开发者)

单张生成 (POST / GET)

调用 /api/generate。支持 JSON 传参或 Form-Data（若需上传 Logo）。

curl -X POST http://localhost:3000/api/generate \
  -H "Content-Type: application/json" \
  -d '{"text": "https://example.com", "size": 300}' \
  --output qrcode.png

批量生成 (POST)

调用 /api/generate-batch，传入 texts 字符串数组，返回 ZIP 文件流。

curl -X POST http://localhost:3000/api/generate-batch \
  -H "Content-Type: application/json" \
  -d '{"texts": ["https://google.com", "Hello"]}' \
  --output batch.zip

QR Code Studio - User Guide

Welcome to QR Code Studio, a modern web app for generating highly customizable QR codes with powerful REST API support.

1. Basic Operations

  • Single Generation: Paste a URL or text into the box, click generate, and preview it on the right. You can then download or copy the image directly to your clipboard.
  • Batch Generation: Input multiple lines of text. The system will generate a QR code for each line and package them into a single .zip archive for 1-click download.
  • Customization: You can adjust the pixel size, use the color pickers for foreground/background colors, and upload a custom Logo to embed in the center.

2. Pro Tip: WiFi Auto-Connect QR Codes

You can generate a QR code that automatically connects smartphones to your WiFi by entering this exact format:

WIFI:S:YourNetworkName;T:WPA;P:YourPassword;;

Note: It must end with two semicolons ;;

3. Excel IMAGE Formula Integration

In modern Excel (Office 365 / 2024), you can render QR codes directly from cell contents (e.g., A1):

=IMAGE("http://localhost:3000/api/generate?text=" & ENCODEURL(A1))

To use custom colors (e.g., blue), escape the # symbol as %23:

=IMAGE("http://localhost:3000/api/generate?size=200&colorDark=%230000FF&text=" & ENCODEURL(A1))

4. API Documentation (For Developers)

Single Generation (POST / GET)

Call /api/generate. Supports JSON or Form-Data (required if uploading a logo).

curl -X POST http://localhost:3000/api/generate \
  -H "Content-Type: application/json" \
  -d '{"text": "https://example.com", "size": 300}' \
  --output qrcode.png

Batch Generation (POST)

Call /api/generate-batch with a texts array to receive a ZIP archive.

curl -X POST http://localhost:3000/api/generate-batch \
  -H "Content-Type: application/json" \
  -d '{"texts": ["https://google.com", "Hello"]}' \
  --output batch.zip
